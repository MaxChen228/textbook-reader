# 譯名規範 — Linear Algebra and Its Applications (Strang)

## 通用原則

- 譯為**台灣繁體中文（zh-TW）**，用台灣慣用譯詞（如「程式」非「程序」、「資料」非「数据」、「演算法」非「算法」）。
- 數學公式、LaTeX、變數符號（$x$, $A$, $\lambda$ 等）**完全保留原樣，不翻譯、不調整空白**。
- 圖／表標號（Figure 1.2, Table 3.1）保留英文 + 編號；圖説內文翻譯。
- 引用書名、人名（Gauss、Pythagoras）保留原文，必要時可在首見處加註中文。
- 標點：句末「。」、列舉「，」、書名《》、引號「」。
- 不要加譯註、不要意譯、不要省略；逐句對應原文資訊量。

## 慣用句

- "if and only if" → 「若且唯若」
- "such that" → 「使得」
- "with respect to" → 「對於」（依語境）
- "Note that" → 「注意到」
- "It follows that" → 「因此」/「由此可得」
- "Proof." → 「證明。」
- "Theorem / Lemma / Corollary / Definition / Example / Remark" → 「定理／引理／推論／定義／範例／註」
